# UI_ISSUES_AND_FIXES_R16.md

## 分级规则

- P0：导致系统不可用、流程主线无法执行、生产数据风险。
- P1：冻结业务规则错误、关键页面阻断用户完成核心流程。
- P2：明显 UI/中文化问题、影响 UAT 观感但不阻断流程。
- P3：测试稳定性、选择器、轻微体验或后续优化项。
