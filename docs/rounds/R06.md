# R06：前端骨架与核心业务页面

## 轮次目标
完成可推进业务主线的前端最小闭环，包括项目列表、项目详情、节点表单、附件上传、第 12 步专用评审页等。

---

## 本轮输入
必须阅读：

- `AGENTS.md`
- `docs/EXECUTION_LEDGER.md`
- `docs/rounds/R06.md`
- 《PRD功能说明书》
- R05 的 API 文档

---

## 本轮范围
允许修改：

- 前端路由与页面
- API client
- 登录页/工作台
- 项目列表页
- 项目详情页
- 节点详情页
- 节点表单区
- 附件上传组件
- 第 12 步专用评审页
- 前端基础测试

本轮不做：

- 大规模图表/可视化
- 完整报表系统

---

## 必做事项

1. 实现登录页或基础鉴权入口。
2. 实现首页工作台。
3. 实现项目列表和详情页。
4. 实现节点详情、表单保存、完成操作。
5. 实现附件上传组件。
6. 实现第 12 步“通过 / 不通过”专用交互。
7. 支持查看轮次历史。
8. 第 13 步金额自动显示 10000。
