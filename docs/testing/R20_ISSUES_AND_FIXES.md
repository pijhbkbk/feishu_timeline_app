# R20 Issues And Fixes

## 1. 分级规则

| 等级 | 定义 | 处理要求 |
|---|---|---|
| P0 | 页面打不开、登录失败、项目无法创建、核心流程无法推进、数据库错误 | 必须立即修复并重跑全量测试 |
| P1 | 关键业务规则错误、权限越权、第 12 步退回失败、第 13 步金额错误、第 17 步未生成 12 个月、材料上传失败 | 必须修复并重跑相关测试和回归 |
| P2 | 中文化不完整、时间线不清晰、数据中心统计缺项、长期加载、明显 UI 错位 | 本轮尽量修复，至少修复影响使用的问题 |
| P3 | 文案、间距、图标、轻微体验问题 | 可记录为后续优化 |
